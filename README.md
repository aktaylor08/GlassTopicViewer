ROS Topic Viewer	
=======

This application allows a user to view information being published in ROS on a user selected topic.

On startup the application will provide the user with a list of currently published topics from which to choose.  It will than let the user select the topic.   After selection the user the application will display the full messages content on screen.

The user also has the option to display information in the topic in the following formats
-- Whole Message
-- Single Field
-- Graphical Representation

This information is displayed until quit is chosen from the menu or until a new topic is selected for viewing.  

## Setup

For setup information see the ROS wiki page.
